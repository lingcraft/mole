# 摩尔抓包工具

## 功能

查看摩尔庄园客户端的解密后的发送包和接收包，并解析发送的通信号、包的命令号、包具体功能、包体数据。点击每列在发送区可显示相应的数据，若点击类型或通信号列，会显示包发送到的服务器地址和端口。
