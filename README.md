# 摩尔抓包工具

## 抓包

可以查看摩尔庄园客户端的解密后的发送包和接收包，并解析发送的通信号、包的命令号、包具体功能、包体数据。点击每列在发送区可显示相应的数据，若点击类型或通信号列，会显示包发送到的服务器地址和端口。拓展发送窗口可指定服务器地址和端口，来实现向非主服务器的小游戏服务器发送数据包。

## 摩摩怪

一键挑战Boss、副本和好友。所有挑战次数均根据当前能量、活力和所有Boss已挑战次数进行计算。一键挑战Boss：挑战每日的3种Boss，超级Boss可自己选择，默认萨尔冥王；一键挑战选定副本；一键挑战好友：师徒优先，然后再按等级从高到低挑战剩余好友，挑战完毕后自动对好友祝福、呼唤、抱抱和解救。

## 拉姆变身值

让当前携带拉姆一键获取每日变身值到卡BUG的上限。

## 元素骑士

一键挑战。若最低攻击力到达7000或者无尽深渊最高记录层数达到50层，并且在副本开放时间内，就执行每日一键挑战无尽深渊70次+莎士摩亚40次+恢复体力+领取每日任务奖励+剩余体力挑战指定副本，若未达到挑战的战力要求，就根据当前剩余体力挑战指定副本。

## 魔灵传说

一键挑战、一键出售。一键挑战希望之光5副本，若在双倍时间之前执行，根据当前体力，计算双倍时间开始时保留至少60体力的最多可挑战次数并进行挑战，竞技场也根据今日剩余次数进行挑战；一键出售魔灵，当前所有队伍已出战魔灵、烈焰剑齿虎和等级大于1级的不会出售，每个数据包出售10个魔灵。

## 点点豆

一键刷取点点豆。

## 餐厅

一键卖菜、自动收菜。一键卖菜：卖餐厅中的当前已选中的餐台的菜，并且价格按最高的焦糖拉姆布丁（50摩尔豆）；自动收菜：根据选定餐台的菜，自动做菜、改菜和收菜，只对灶台上的阳光酥油肉松或酱爆雪顶菇生效。由于摩尔庄园的历史遗留问题，导致餐厅待久了会越来越卡，所以目前只能做一点时间短的菜，并且做几个小时候需要刷新下游戏，需要以后开发了协议端餐厅功能才能解决。
